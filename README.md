# Vectorfy

Interactive Math game where you will learn all about 3D and 2D vectors.

# Controls

Select "Draw" to draw vectors
Select "Erase" and click on a vector you want to erase
Select "Select" Click on a vector you want to select or RMB + shift to select more
After you select vector you can add them, multiply them
